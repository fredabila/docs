<Note>
**Access Path**: **Event Dashboard** > **Engage** > **Page Builder**
</Note>

The **Enhanced Page Builder** allows you to go beyond standard layouts and design a completely custom mobile landing page for your event. Use the drag-and-drop interface to build a page that matches your brand perfectly.

## The Interface

- **Component Library** (Left): The toolbox containing all available building blocks, categorized by type.
- **Canvas** (Center): The live mobile preview area where you drag and drop components.
- **Controls** (Top): Toggle between **Edit** and **Preview** modes, and switch device views (Mobile/Tablet/Desktop).

## Component Library

### Layout
Structure your page basic building blocks.
- **Section**: A full-width background area.
- **Container**: A box to hold content, useful for card-like designs.
- **Columns**: Split content side-by-side.
- **Spacer** & **Divider**: Add breathing room and separation.

### Content
Add text and calls to action.
- **Hero**: A large header image with overlay text.
- **Heading** & **Text**: Standard typography tools.
- **Button**: Create links to external sites or actions (e.g., "Get Tickets").

### Media
- **Gallery**: Display a carousel of images.
- **Video**: Embed promotional videos.
- **Testimonial**: Showcase social proof and reviews.

### Dynamic
Automatically pull in data from your event settings.
- **Event Info**: Displays date, time, and location automatically.
- **Sponsors**: Renders your sponsor logos grid.
- **Sessions**: Shows the agenda/schedule.
- **Organizer**: Displays the organizer profile.

### Advanced
- **HTML**: Insert custom code snippets.
- **Rich Text**: A versatile editor for formatted content.

## Publishing

Once your design is ready:
1.  Click **Save Layout** in the top right.
2.  Use the **Enable Custom Page** toggle in the bottom left **Visibility Control** panel to make the page live for attendees.
